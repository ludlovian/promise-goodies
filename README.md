# promise-goodies
Single idempotent default function export which adds loveliness to `Promises`. If you don't specify the promise library, it monkey patches the global one. So, sue me.

## Goodies added

- `all`
- `catchif`
- `deferred`
- `delay`
- `finally`
- `isResolved`
- `map`
- `race`
- `reject`
- `resolve`
- `timeout`
